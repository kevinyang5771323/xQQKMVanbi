# 大学生请假管理系统 python842

## 项目概述

大学生请假管理系统是一款基于Python和Django开发的网络应用程序，旨在简化和数字化高校中的请假流程。系统采用B/S架构，前端使用Vue.js技术，后端使用Django框架，数据库采用MySQL，并集成了数据预测算法以优化管理功能。

## 功能模块

### 学生角色

- 个人中心
 - 管理个人信息
- 学生请假
 - 提交请假申请

### 辅导员角色

- 个人中心
 - 管理个人信息
- 学生请假
 - 审核学生请假申请
- 班级信息
 - 管理班级信息
- 请假数据
 - 查看请假数据统计
- 请假次数预测
 - 预测学生请假次数

### 管理员角色

- 个人中心
 - 管理个人信息
- 学生管理
 - 管理学生信息
- 辅导员管理
 - 管理辅导员信息
- 班级管理
 - 管理所有班级
- 请假审核
 - 审核学生请假申请
- 请假数据统计
 - 分析请假数据
- 请假次数预测
 - 预测整体请假趋势
- 系统设置
 - 配置系统参数

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL
- 系统架构：B/S结构
- 特殊功能：数据预测算法
